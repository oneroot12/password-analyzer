# Password Analyzer 🔐

Analyzes password strength and estimates brute-force crack time.

## Features
- Strength: Weak / Medium / Secure
- Detects missing elements (letters, numbers, symbols)
- Estimates crack time (seconds → years)

## Usage
Run the script and enter a password.

## Example
Strength: Medium
Reasons:
- Missing symbols
